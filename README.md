## Pinbelegung  

### LED-Matrix  
- **VCC** = 5V (Breadboard)  
- **GND** = GND (Arduino)  
- **DOUT (DIN)** = 13  
- **CS** = 12  
- **CLK** = 11  

### Joystick  
- **VCC** = 5V (Arduino)  
- **GND** = GND (Arduino)  
- **VRx** = A0  
- **VRy** = A1  
- **SW** = 10  

### Power-Supply-Modul  
- **5V** = Vin  
- **5V** = Breadboard  
- **GND** = GND (Arduino)

# Documentation

... is an Arduino-based retro console.

## Components

- Arduino Uno R3  
- Joystick module  
- 8×8 LED matrix (monochrome)  
- 170-pin breadboard  
- Power supply module  
- 9V block battery  

## STL Files  
[Thingiverse Link](https://www.thingiverse.com/thing:6975936/files)  

# Code  
In the project folder, you will find both the complete PlatformIO project and the individual, ready-to-use games (if all pins are correctly connected).  

## Pin Layout  

### LED Matrix  
- **VCC** = 5V (breadboard)  
- **GND** = GND (Arduino)  
- **DOUT (DIN)** = 13  
- **CS** = 12  
- **CLK** = 11  

### Joystick  
- **VCC** = 5V (Arduino)  
- **GND** = GND (Arduino)  
- **VRx** = A0  
- **VRy** = A1  
- **SW** = 10  

### Power Supply Module  
- **5V** = Vin  
- **5V** = breadboard  
- **GND** = GND (Arduino)
